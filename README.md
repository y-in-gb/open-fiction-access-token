# Open Fiction Access Token (OFAT)

查看中文版 README -> [`README-zh.md`](README-zh.md)

This repository contains **OFAT v1.0**, a custom open license for the fiction [潮篡 Over Flow And Throw] written by me, the original author [崖雁 yayan].

---

## 🎟️ The Tokens

No waiting. No hidden fees. The OFAT v1.0 license gives you specific **Tokens**(free passes) to build upon the fiction:

| Your Token | What You Can Build | Commercial-Use
|------------|---------------------|--------------|
| 🔬 **Research** | Model training/fine-tuning, GraphRAG, linguistics, and other research purpose covered by the license | See [`LICENSE`](LICENSE) |
| 🤖 **AI Media** | AI-generated videos, animations, webcomics | ✅ |
| ✍️ **Text Adaptations** | Translation, fan fiction, rewritten endings, spin-off stories | ✅ |
| 🎨 **Human Art** | Illustrations, character designs, visual interpretations | ✅ |
| 📖 **Share** | Copy and distribute the fiction as digital or printed book | ✅ |

*Note: Please follow the corresponding license terms in [`LICENSE`](LICENSE) as you use the tokens*
---

## 🚀 How OFAT v1.0 Works

1. 📝 **The author** publishes the Work under OFAT v1.0 license
2. 🎟️ **You** pick a token (Share, Text Adaptations, AI Media, Human Art, or Research)
3. ✅ **You follow the license terms** — See [`LICENSE`](LICENSE)
4. 🛠️ **You build** freely and legally
5. 📤 **You share back to the community** - Research or Adaptations
